## 🤝 Cara Berkontribusi

Kami sangat menyambut kontribusi dari developer lain! Berikut adalah langkah-langkah untuk berkontribusi pada proyek open source ini:

1. **Fork Repositori**: Klik tombol 'Fork' di pojok kanan atas halaman ini untuk menyalin repositori ke akun GitHub Anda.
2. **Klon Repositori**: Unduh proyek ke mesin lokal Anda.
   ```bash
   git clone https://github.com/username-anda/app-kawanua.git
   ```
3. **Install Dependensi**: Pastikan Anda telah menginstal Node.js, lalu jalankan:
   ```bash
   npm install
   # atau jika menggunakan pnpm
   pnpm install
   ```
4. **Buat Branch Baru**: Gunakan nama branch yang deskriptif untuk fitur atau perbaikan Anda.
   ```bash
   git checkout -b feature/nama-fitur-anda
   ```
5. **Lakukan Perubahan**: Implementasikan fitur atau perbaikan bug Anda. Pastikan kode Anda mengikuti standar linting yang ada.
6. **Commit Perubahan**: Simpan perubahan Anda dengan pesan commit yang jelas.
   ```bash
   git commit -m "feat: Menambahkan fitur X yang baru"
   ```
7. **Push ke GitHub**: Kirim perubahan Anda ke repositori fork Anda.
   ```bash
   git push origin feature/nama-fitur-anda
   ```
8. **Buka Pull Request**: Buka halaman repositori asli dan ajukan Pull Request dari branch Anda. Kami akan meninjau kontribusi Anda secepat mungkin!
